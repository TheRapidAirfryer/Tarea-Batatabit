# Batatabit
Landing page responsiva para batatabit, una plataforma de criptomonedas.

## Descripción
Batatabit es una plataforma de criptomonedas que permite a los usuarios comprar, vender y almacenar criptomonedas.

## Características
- **Información en tiempo real**: La tasa se actualiza minuto a minuto
- **Transparencia Total**: Sin tasas ocultas en compra o venta

## Tecnologías Utilizadas
- HTML5
- CSS3
- Flexbox
- CSS Grid

## Instalación
1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.
